Concurrent

Whirlwind allows for fluid, thread-safe, manageable concurrency through the use of implicit locking and asynchronous functions.

Functional

Whirlwind also has features to allow for functional programming, such as comprehensions and aggregators.

Expressive

Whirlwind presents a legible, simple expression syntax, while still allowing for a myriad of useful operators.

Object Oriented

Whirlwind brings a new kind of object orientation to the table, while still remaining simple and efficient.

Concise Flow

Whirlwind introduces a number of powerful control flow operators to make your code cleaner.
