Concurrent

Whirlwind allows for fluid, thread-safe, manageable concurrency through the use of locking and asynchronous functions.

Functional

Whirlwind also has features to allow for functional programming, such as comprehensions and aggregators.

Generic

Whirlwind provides a powerful generic typing system to allow for efficient and dynamic programming.

Object Oriented

Whirlwind brings a new kind of object orientation to the table, while still remaining simple and efficient.
