Language Documentation

Getting Started Guide

This is a guide designed to get you started up and ready to go with Whirlwind. It is by no means comprehensive, but it will give you enough information to build a good understanding of the language.

Syntax Reference

This is the formal language grammar and list of lexical tokens. It is marked up in a modified EBNF format and is a copy of the actual grammar and tokens used by the compiler.

Language Specification

This is the formal language specification of Whirlwind. It is not an easy read, but definitely worth it for any advanced Whirlwind developers.

Style Guide

This is the official Whirlwind style guide. It describes naming conventions, formatting, and how to write efficient and idiomatic Whirlwind. An absolute must read for Whirlwind programmers new and old alike.

Blend Documentation

Introducing Blend

This is an article designed to get you up to speed with Whirlwind's builtin project management tool, Blend. While it is not comprehensive, it is very detailed, but still fairly friendly.

Blend Reference

This is the complete CLI and concept reference for Blend. It gives a list of all commands and their syntax, and it provides a full terminology and concept guide.

Package Documentation

Standard Library Reference

This is a searchable reference for all package in the standard library. It not only describes their API, but also how to use them and what they do. This will likely be one of you most valuable resources when working with Whirlwind.

Working with Packages

This document describes how to create, install, manage, and publish Whirlwind packages you and others have created. If you plan on contributing to or benefiting from Whirlwind's package ecosystem, this is an essential read.
