Whirlwind

A lightning fast, intuitive systems language for the modern world.

Why Whirlwind?

Whirlwind is a language designed to resolve many of the problems of other languages. It is intended to bridge the gap between the more hardcore systems languages such as C++ or Rust and languages like Python and Ruby, while still remaining efficient. While this task sounds impossible, Whirlwind has come as close as possible to being this bridge. It is an imperative language with functional aspects that introduces the unique tool of agents. It is also designed to be expressive and user friendly, while simultaneously giving you extreme power and tight control. In short, it is a language designed to fill the gaps and minimize the problems and idiosyncracies of many other languages. It is not perfect, but a good start.
